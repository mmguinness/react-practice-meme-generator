## freeCodeCamp 

# Meme Generator

Practice project as part of the freeCodeCamp course - [React JavaScript Library 2022](https://www.youtube.com/watch?v=bMknfKXIFA8)


## Tech Used

- JavaScript
- ReactJS
- HTML
- CSS
- Meme images from the [imgflip API](https://api.imgflip.com/get_memes)


## Requirements    

### Brief

- Tutorial to make a dynamic application in React that pulls the most popular meme images from imgFlip and allows user to add their own text.
